What are semantic tags in html, and why are they important?

Semantics are tags that are used in coding to make it easier to read online, they help devices to determine important content in the code to display and its context on the site. Some examples are article,details,and footer

What is the difference between a block-level element and an inline element?

A block-leavel element is an element that always takes up the full width available, and it also starts a new line that with give it margine before and after. An inline element on the other hand does not start a new line of code, and it only takes up the space necessary for the code.
